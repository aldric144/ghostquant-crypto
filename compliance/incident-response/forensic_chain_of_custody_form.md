## Completion Instructions

### Required Fields

**All Evidence Must Include**:
1. Evidence ID (unique identifier)
2. Collection Date/Time (UTC)
3. Collector Name and Signature
4. Source Engine or System
5. Detailed Evidence Description
6. SHA-256 Hash Value
7. Initial Storage Location
8. Genesis Archive™ Block Reference (if applicable)

### Transfer Requirements

**Every Transfer Must Document**:
1. Transferor name and signature
2. Recipient name and signature
3. Transfer date/time (UTC)
4. Transfer purpose
5. Evidence location after transfer

### Access Requirements

**Every Access Must Document**:
1. Accessor name and signature
2. Access date/time (UTC)
3. Access purpose
4. Actions performed
5. Access duration

### Integrity Verification Requirements

**Verification Must Occur**:
1. Upon initial collection
2. Before every transfer
3. After every transfer
4. Before any access
5. Monthly for active investigations
6. Annually for archived evidence

## Digital Signature Requirements

### Electronic Signatures

**Acceptable Electronic Signatures**:
- Digital signature with PKI certificate
- Biometric signature capture
- Multi-factor authenticated signature
- Blockchain-anchored signature (Genesis Archive™)

**Signature Requirements**:
- Unique to signer
- Verifiable identity
- Tamper-evident
- Audit trail maintained
